## 세션 1: 프로그램이란? / 왜 파이썬인가? (★★★★★)

### 📖 이론 (10분)

#### 1.1 프로그램이란?
- **정의:** 컴퓨터에게 내리는 명령어들의 집합
- **실생활 비유:** 요리 레시피, 조립 설명서와 같은 순차적 지시사항

#### 1.2 왜 프로그래밍을 배워야 할까?

**💼 실무에서의 효과**
- **시간 절약:** 30분 걸리던 엑셀 작업 → 3초로 단축
- **정확성:** 사람의 실수 0%로 감소
- **자동화:** 반복 작업을 컴퓨터에게 맡기고 창의적인 일에 집중

**📊 시장 수요 (2024년 기준)**
- 개발자 평균 연봉: 신입 4,500만원, 경력 5년 7,000만원+
- 비개발자도 Python 능력 시 연봉 10-20% 상승
- 전 세계 채용 공고 중 Python 요구 비율: 35% (1위)

**🎯 실제 활용 사례**
- **마케터:** 매일 아침 자동으로 광고 성과 리포트 생성 및 이메일 발송
- **인사팀:** 100명의 급여명세서를 5초만에 자동 생성
- **영업팀:** 고객 데이터에서 구매 패턴 자동 분석

#### 1.3 왜 파이썬인가?

**📈 통계로 보는 파이썬**
- **Stack Overflow 2024:** 가장 배우고 싶은 언어 1위 (7년 연속)
- **GitHub:** 전 세계에서 2번째로 많이 사용되는 언어
- **Google:** 공식 3대 언어 중 하나 (C++, Java, Python)

**✨ 파이썬의 강점**

**1) 압도적으로 쉬운 문법**

> **💡 Java vs Python 비교**
>
> 같은 기능을 구현하는데 필요한 코드 양 비교

```java
// Java - "Hello, World!" 출력
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
// 5줄 필요
```

```python
# Python - "Hello, World!" 출력
print("Hello, World!")
# 1줄로 끝!
```

**2) 빠른 개발 속도**
- Java로 1주일 걸릴 프로젝트 → Python으로 1-2일
- 아이디어를 즉시 코드로 구현 가능

**3) 만능 언어**
- 웹 개발, 데이터 분석, AI, 자동화, 게임... 안 되는 게 없음
- 한 언어만 배우면 여러 분야 진출 가능

**4) 방대한 라이브러리**
- 50만개 이상의 무료 라이브러리 (누가 이미 만들어놨음!)
- 엑셀 처리, 웹 크롤링, AI 모델까지 코드 3줄이면 해결

#### 1.4 파이썬으로 할 수 있는 실제 예시

**예시 1) 엑셀 파일 100개를 1초만에 합치기**
```python
import pandas as pd
import glob

# 모든 엑셀 파일 찾기
files = glob.glob("*.xlsx")

# 한 번에 합치기
result = pd.concat([pd.read_excel(f) for f in files])

# 하나의 파일로 저장
result.to_excel("통합본.xlsx", index=False)

# 실행 시간: 1초
# 수동으로 하면: 30분+
```

**예시 2) 네이버에서 뉴스 제목 자동으로 가져오기**
```python
import requests
from bs4 import BeautifulSoup

# 네이버 뉴스 페이지 가져오기
response = requests.get("https://news.naver.com")
soup = BeautifulSoup(response.text, 'html.parser')

# 뉴스 제목 추출

# 출력

    print(title.text)

# 100개 뉴스 제목 수집: 3초
# 수동으로 복사하면: 10분+
```

**예시 3) 폴더 안 파일 1000개 이름 한 번에 바꾸기**

## 세션 2: 파이썬 특징과 활용 분야 (★★★★)

### 📖 이론 (10분)

#### 2.1 파이썬의 주요 특징

**1) 인터프리터 언어**

> **📌 개념: 인터프리터란?**
>
> 코드를 한 줄씩 바로 실행하는 방식
> - **컴파일 언어 (Java, C++):** 전체 코드를 기계어로 번역 → 실행
> - **인터프리터 언어 (Python):** 한 줄씩 번역하면서 바로 실행
>
> **장점:** 코드 수정 후 바로 결과 확인 (빠른 개발)
> **단점:** 실행 속도는 컴파일 언어보다 느림 (하지만 대부분의 경우 문제 없음)

**Java vs Python: 실행 과정 비교**
```
[Java]
코드 작성 → 컴파일(javac) → .class 파일 생성 → JVM 실행
시간: 코드 수정 후 결과 확인까지 30초~1분

[Python]
코드 작성 → 즉시 실행 (python)
시간: 코드 수정 후 결과 확인까지 1초
```

**2) 동적 타입 언어**

> **📌 개념: 동적 타입이란?**
>
> 변수의 타입을 자동으로 판단하는 방식
> - **정적 타입 (Java):** 변수 선언 시 타입을 미리 지정
> - **동적 타입 (Python):** 변수에 값을 넣으면 자동으로 타입 결정

**Java vs Python: 변수 선언 비교**
```java
// Java - 타입을 명시해야 함
String name = "김파이썬";
int age = 28;
double height = 175.5;

// 타입이 다르면 오류
int number = "123";  // ❌ 컴파일 오류!
```

```python
# Python - 타입 자동 판단
name = "김파이썬"  # 알아서 문자열로 인식
age = 28          # 알아서 정수로 인식
height = 175.5    # 알아서 실수로 인식

# 타입 변경도 자유롭게
number = 123      # 정수
number = "123"    # 문자열로 변경 가능 ✅
```

**3) 들여쓰기 문법**

> **📌 개념: 들여쓰기 (Indentation)**
>
> 코드 블록을 들여쓰기로 구분하는 방식
> - **대부분 언어:** 중괄호 {} 사용
> - **Python:** 들여쓰기(스페이스 4칸 또는 탭) 사용
>
> **장점:** 코드가 자동으로 깔끔해짐 (읽기 쉬움)

**Java vs Python: 조건문 비교**
```java
// Java - 중괄호 사용
if (age >= 18) {
    System.out.println("성인입니다");
    System.out.println("투표 가능");
}
```

```python
# Python - 들여쓰기 사용
if age >= 18:
    print("성인입니다")
    print("투표 가능")
```

#### 2.2 파이썬 활용 분야와 실제 예시

**1) 데이터 분석 & 시각화**
- **사용처:** 매출 분석, 고객 데이터 분석, 통계 리포트
- **주요 기업:** Netflix (추천 시스템), NASA (우주 데이터 분석)

**실제로 할 수 있는 것:**
```python
import pandas as pd
import matplotlib.pyplot as plt

# 엑셀 데이터 읽기
sales = pd.read_excel("월별매출.xlsx")

# 월별 매출 합계
monthly_total = sales.groupby("월")["매출"].sum()

# 그래프 그리기
monthly_total.plot(kind='bar')
plt.title("월별 매출 현황")
plt.savefig("매출그래프.png")

# 실행 시간: 5초
# 엑셀로 하면: 10분+
```

**2) 웹 개발**
- **사용처:** 웹사이트, 웹 애플리케이션, API 서버
- **주요 기업:** Instagram, Spotify, YouTube, Dropbox

**3) 인공지능 & 머신러닝**

**4) 자동화 & 크롤링**
- **사용처:** 데이터 수집, 업무 자동화, 테스트 자동화

>
> **왜 필요한가?**
> ```
> 상황:
> - 프로젝트A는 pandas 버전 1.0 필요
> - 프로젝트B는 pandas 버전 2.0 필요
> - 하나의 컴퓨터에 두 버전을 동시에 설치할 수 없음!
>
> 해결:
> - 프로젝트A용 가상환경 → pandas 1.0 설치
> - 프로젝트B용 가상환경 → pandas 2.0 설치
> - 각각 독립적으로 관리!
> ```
>
> **실무에서는 필수!**
> - 협업 시 "내 컴퓨터에서는 되는데요?" 문제 방지
> - 배포 시 필요한 라이브러리 정확히 명시

## 세션 5: print() 함수 (★★★★★)

### 📖 이론 (10분)

#### 5.1 print() 함수란?

> **📌 개념: 함수(Function)란?**
>
> 특정 작업을 수행하는 코드 묶음
> - print() 함수: 화면에 출력

**Java vs Python: 출력 비교**
```java
// Java
System.out.println("Hello");
```

```python
# Python
print("Hello")  # 훨씬 간단!
```

#### 5.2 기본 사용법
```python
# 문자열 출력
print("Hello, World!")

# 숫자 출력
print(100)
print(3.14)

# 여러 값 동시 출력
print("이름:", "김파이썬")
print("나이:", 28, "세")
```

#### 5.3 유용한 옵션
```python
# sep: 구분자 지정
print("2025", "12", "06", sep="-")  # 2025-12-06

# end: 끝 문자 지정
print("안녕하세요", end=" ")
print("반갑습니다")  # 안녕하세요 반갑습니다

# 여러 줄 출력
print("""첫 번째 줄
두 번째 줄
```

## 세션 9: 불린 자료형 (bool) (★★★★)

### 📖 이론 (10분)

#### 9.1 불린(Boolean)이란?

> **📌 개념: 불린 (Boolean)**
>
> 참(True) 또는 거짓(False) 두 가지 값만 가지는 자료형
>
> **중요성:** "프로그램의 흐름 제어의 핵심!"

```python
is_student = True
is_married = False

print(type(True))   # <class 'bool'>
```

**Java vs Python: 불린 타입**
```java
// Java
boolean isStudent = true;   // 소문자
```

```python
# Python
is_student = True  # 대문자!
```

#### 9.2 비교 연산자

```python
print(10 > 5)     # True
print(10 < 5)     # False
print(10 >= 10)   # True
print(10 <= 5)    # False
print(10 == 10)   # True
print(10 != 5)    # True

# 문자열 비교
print("apple" == "apple")  # True
print("Apple" == "apple")  # False
```

> **💡 주의: = vs ==**
>
> ```python
> # = : 할당
> age = 28
>
> # == : 비교
> age == 28  # True
> ```

#### 9.3 논리 연산자

```python
# and: 둘 다 참이어야 참
print(True and True)     # True
print(True and False)    # False

# or: 하나라도 참이면 참
print(True or False)     # True
print(False or False)    # False

# not: 반대로 뒤집기
print(not True)          # False
```

**실용 예시**
```python
age = 25
has_license = True

can_drive = (age >= 18) and has_license
print(can_drive)  # True
```

#### 9.4 Truthy vs Falsy

**Falsy 값 (거짓으로 평가)**
```python
print(bool(0))       # False
print(bool(""))      # False
print(bool(None))    # False
```

**Truthy 값 (참으로 평가)**
```python
print(bool(1))       # True
print(bool("Hi"))    # True
print(bool([1, 2]))  # True
```
